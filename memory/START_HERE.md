# Start Here

## Mission

Maintain an evidence-focused BPC-157 research library that stays clear about the gap between preclinical enthusiasm and human evidence.

## First actions for a new thread

1. Read [session-handoff.md](/Users/terryperkinson/codex/bpc157/memory/session-handoff.md).
2. Read [resource-map.md](/Users/terryperkinson/codex/bpc157/memory/resource-map.md).
3. Validate the catalog with `npm run validate`.
4. Search the library before adding new claims: `npm run search -- <term>`.
5. Update the public site with `npm run build` after any material change.

## Operating rules

- Prefer primary sources and registry records.
- Treat reviews as synthesis, not proof.
- Separate animal findings from human findings every time.
- Update `memory/open-questions.md` when research uncertainty remains.
- Update `memory/session-handoff.md` before ending a work session.
